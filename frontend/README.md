# Food Delivery App - MERN Stack

A full-stack food delivery application similar to Zomato.

## Technologies
- MongoDB
- Express.js
- React.js
- Node.js

## Features
- User Authentication
- Browse Restaurants
- Shopping Cart
- Order Management
- Order History

## Installation

### Backend
```bash
cd backend
npm install
# Create .env file with MONGO_URI, JWT_SECRET, PORT
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm start
```
